## Key Decisions

1. **Multi-step Rust-side filtering** over SurrealQL subqueries for ready-work — simpler, easier to debug, acceptable for small-to-medium task sets.
2. **Backtick escaping** for `type` in SurrealQL WHERE clauses — `type` is a reserved keyword and silently returns 0 rows without escaping.
3. **Conditional datetime casting** using `IF $field != NONE THEN <datetime>$field END` for optional datetime fields in UPSERT.
