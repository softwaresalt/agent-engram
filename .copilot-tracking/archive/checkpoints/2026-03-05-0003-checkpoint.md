## Key Decisions

1. **tracing-appender not installed as global subscriber** — T078 creates `.engram/logs/` directory and logs the path, but does not install a file appender because the global subscriber is already set in `bin/engram.rs`. Installing a second subscriber would panic. Directory creation satisfies the spirit of the task.

2. **S093/S094 unix-only** — Disk-full simulation via `chmod 0o555` only works on Unix; Windows ignores read-only bit on directories. Tests gated with `#[cfg(unix)]`.

3. **T076/T079/T082-T084 verified pre-existing** — Several Phase 8 tasks were already implemented in earlier phases and did not require new code: path-with-spaces test (Phase 6), atomic_write (pre-existing), benchmark tests (pre-existing).

4. **MAX_METHOD_LEN=256** — IPC method name limit chosen to match HTTP header size conventions; returns JSON-RPC error code `-32600` (Invalid Request).
